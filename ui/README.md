# Mira Studio UI

Svelte + Tauri desktop app for Mira.

## Development

```bash
# Install dependencies
npm install

# Run dev server (web only)
npm run dev

# Run Tauri dev (desktop app with hot reload)
npm run tauri dev
```

## Build

```bash
# Build for current platform
npm run tauri build
```

## Stack

- **Frontend**: Svelte 5 + Vite + TailwindCSS
- **Desktop**: Tauri 2
- **Terminal**: xterm.js with Catppuccin theme
- **Styling**: Catppuccin Mocha color scheme
